# Lakefront Premium — Key Data

## Scarcity
- Shoreline: ~49–50 km (LfU)
- Surface area: 56.4 km²; 2nd largest Bavarian lake
- 35–45% of shoreline is Wittelsbach / monastery / state / conservation (off-market perpetually)
- Estimated 900–1,100 direct-lakefront parcels; 620–780 privately tradeable
- Annual turnover: 8–16 direct-lakefront transactions; 2–5 ultra-prime (>€10M)

## Shore tiers
- **Tier 1+ (€28–60k/m² land, up to €70k+)**: Feldafing (Park-edge)
- **Tier 1 (€22–55k/m²)**: Berg/Leoni/Kempfenhausen, Münsing/Ammerland/Ambach, Tutzing, Pöcking/Possenhofen
- **Tier 2 (€15–32k/m²)**: Starnberg, Seeshaupt, Bernried

## Legal regime
- **BayWG §21**: 5m Gewässerrandstreifen (default), often extended to 10–30m via overlay, 40–50m w/ B-Plan
- **BauGB §35**: Lakefront outside B-Plan = Außenbereich, no new building except Bestandsschutz replacement — core reason Altbestand is gold
- **LSG "Starnberger See Ost/West"**: controls tree removal, hedging, fencing, Steg aesthetics
- **Roseninsel**: Bodendenkmal + NSG, untouchable
- **BayDSchG**: Denkmal obligation + §7i/§11b EStG AfA (9%×8yr, 7%×4yr)
- **GEG §105**: Denkmal exempt from most, but internal insulation forced → higher running costs

## Seeuferweg wars
- BayWG Art. 27 grants Betretungsrecht; BayVGH has affirmed designations
- Münsing fought 15+ years; Berg also disputes
- **Value impact**: enforced path through parcel = -20–35% premium; path along driveway -5–10%; "under review" -5–12%

## Private docks (Stege)
- WWA Weilheim grants essentially no new private Stege; new only as Bestandsersatz
- Altbestand private Steg w/ Wasserrecht: +€150–450k value
- Bootshaus w/ Wasserrecht: +€800k–€2.5M

## Hedonic premium
- Academic lake literature: direct-lakefront = 3–6x inland
- Starnbergersee empirically: 6–10x hillside-view in same Gemeinde; 15–25x inland no-view
